Welcome to the store, now find the most expensive items. The first number you are given is the quantity of expensive items we want to find. The store's inventory is then represented as a list of {% if interpreter.slug == "js-node" %}objects{% else %}dicts{% endif %}

{% if interpreter.slug == "js-node" %}

Input: Integer and array of objects. Each objects has two attributes "name" and "price"

{% else %}

Input: int and list of dicts. Each dict has two keys for the "name" and "price" of each item.

{% endif %}

Output: A list of the complete {% if interpreter.slug == "js-node" %}object{% else %}dict{% endif %} for the number of expensive items requested.

Example:

{% if interpreter.slug == "js-node" %} 
biggerPrice(2, [
    {"name": "bread", "price": 100},
    {"name": "wine", "price": 138},
    {"name": "meat", "price": 15},
    {"name": "water", "price": 1}
]) == [
    {"name": "wine", "price": 138},
    {"name": "bread", "price": 100}
]

biggerPrice(1, [
    {"name": "pen", "price": 5},
    {"name": "whiteboard", "price": 170}
]) == [{"name": "whiteboard", "price": 170}]
